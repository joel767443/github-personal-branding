Step 1 of 2

GitHub token setup

Save your GitHub Personal Access Token and your profile repository URL so sync can run with your own credentials.

How to get your GitHub personal access token

  1. Open GitHub and go to Settings -> Developer settings -> Personal access tokens -> Fine-grained tokens.
  2. Create a new token with at least read access to repositories you want to sync.
  3. Copy the token now, then paste it below. GitHub only shows the full token once.

How to get your profile repository URL

  1. Open the repository that represents your profile/deployment source on GitHub.
  2. Click Code and copy the HTTPS clone URL.
  3. Use the full URL format like https://github.com/username/repo.git.
Step 2 of 2

LinkedIn export upload

After saving your GitHub token, upload your LinkedIn data export ZIP to continue. Once uploaded, LinkedIn processing and GitHub repo sync will run automatically in the background.

How to export your LinkedIn ZIP

  1. Sign in to LinkedIn, open Settings & Privacy, then go to Data privacy.
  2. Click Get a copy of your data.
  3. Select Download larger data archive (recommended), or select all categories, then submit the request.
  4. Wait for LinkedIn's email that your archive is ready, then click Download archive.
  5. Save the downloaded .zip file and upload the same file below (do not unzip it).

Enter LinkedIn API credentials first. The access token is encrypted at rest.
