GitHub token setup

Save your GitHub Personal Access Token and your profile repository URL so sync can run with your own credentials.

How to get your GitHub personal access token

  1. Open GitHub and go to Settings -> Developer settings -> Personal access tokens -> Fine-grained tokens.
  2. Create a new token with at least read access to repositories you want to sync.
  3. Copy the token now, then paste it below. GitHub only shows the full token once.

How to get your profile repository URL

  1. Open the repository that represents your profile/deployment source on GitHub.
  2. Click Code and copy the HTTPS clone URL.
  3. Use the full URL format like https://github.com/username/repo.git.
